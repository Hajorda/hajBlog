# Aviation Emissions Control Dashboard ✈️🌍

A state-of-the-art, premium web application designed to track and visualize network-wide aviation sustainability metrics and CO₂ emissions. Built with a focus on high-end UI/UX (glassmorphism, micro-animations, and dynamic data visualization).

## Key Features
- **Real-time Sustainability Tracking:** Monitor 7-day total CO₂ metrics and overall flight volumes across the network.
- **Interactive Filtering:** Slices and dice data using interactive dropdowns to filter by specific Airlines or historical Date Ranges.
- **Advanced Visualizations:** Uses Recharts for dual-axis charts, scatter plots, radar charts, and congestion heatmaps.
- **Premium UX:** Built with React 18, Tailwind CSS v3, and Framer Motion for a deeply interactive experience.
